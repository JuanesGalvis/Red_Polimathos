PREGUNTAS FRECUENTES

Nuestra prioridad son nuestros creadores, por ende buscandos que soluciones tus preguntas sobre la plataforma

Qué es Polimathos?

Es una plataforma tipo red social con perfiles con diferentes aspectos, dependiendo del ámbito en el que se desenvuelven. Además, secciones tipo foro para discutir y colaborar sobre diversos temas. No nos enfocamos en la especialización sino en aprender de TODO!

Cómo se crea mi @usuario?

El sistema de creación de usuarios de Polimathos genera el @usuario mediante las dos primera palabras del nombre (Ejm: Carlos Jose -> @Carlos_Jose)

Polimathos es solo para progradores?

Polimathos es para todo aquel que cree contenido, puede ser en código (programación), arte (diseño), vídeo (audiovisual) o contenido en general (creador de contenido)

Qué son los badge del perfil?

Son las tecnologías que dominas o quieres que otros sepan que manejas

Qué son las discusiones?

Si tienes dudas o tema para discutir con la comunidad, puedes crear o buscar el tema y crear tu discusión

Puedo borrar mi cuenta?

Preferiblemente no, pero claro que puedes, desde Configuración

Donde puedo reportar un problema?

Puedes hacerlo desde Configuración

Cómo esta desarrollado Polimathos?

Realizado con HTML, CSS y JavaScript Vanilla (Ningun framework) y algunos estilos con preprocesador Sass.

Puedo colaborar con el proyecto?

Claro que si!, este es el repositorio

DOCUMENTACIÓN

Polimathos maneja HTML nativo (Sin preprocesadores) y CSS con algunos estilos en el preprocesador Sass, las clases son nombradas mediante BEM:

.BLOQUE__ELEMENTO--MODIFICADOR
Mediante una paleta de colores mantenemos los colores de la marca iguales en las pantallas:
  • Azul: #3540A5
  • Morado: #B8B8D1
  • Amarillo: #FFC145
  • Blanco: #FFFFFB
  • Rojo / Rosa: #FF6B6C

También se utilizó el framework Bootstrap para algunos estilos
